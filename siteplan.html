Site Name

SkyBoard

I chose "SkyBoard" because it sounds modern and cool, like skateboard but for board games. It's short and easy to remember. Plus it works good in English and Georgian people can say it easy too.

Optional domain: skyboard.ge

Site Purpose

The SkyBoard website is for board game lovers in Tbilisi. The site will have:

  • Game catalog with basic info about games
  • Booking system for tables
  • Information about events and tournaments
  • Some game rules and beginner tips

We want to make board gaming more popular in Tbilisi and give people a place to play and meet others.

Scenarios

Scenario 1

Question: "What games do you have for kids and can I book a table for Saturday?"

Scenario 2

Question: "I never played board games before. What easy games you recommend?"
